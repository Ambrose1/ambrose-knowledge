# 第五章：致富 vs 守富（Getting Wealthy vs. Staying Wealthy）

## 核心观点

**致富和守富是两种完全不同的技能。** 致富需要冒险、乐观、敢于押注；守富需要谦逊、恐惧、承认运气。

> 大多数人擅长其中一种，极少有人两者兼备。而财富最终的归宿，取决于你守不守得住。

## 关键洞见

### 1. 两种技能的反向要求

| | 致富 | 守富 |
|------|------|------|
| **心态** | 乐观主义 | 谨慎悲观 |
| **策略** | 集中押注 | 广泛分散 |
| **对风险的态度** | 敢冒风险 | 恐惧风险 |
| **时间尺度** | 抓住某个时刻 | 穿越所有时刻 |
| **核心能力** | 勇气 | 谦逊 |

> 同一个人很难在两种模式下自由切换。致富后继续用致富的思维方式——结果就是守不住。

### 2. 书中的例子

- **Jesse Livermore**：1929 年做空赚了相当于今天 30 亿美元的财富，但继续用高杠杆博更大的 → 最终破产自杀
- 他致富的技能和守富的技能是同一套——这在致富时是优点，在致富后是致命缺陷

### 3. 守富的本质

> 守富不是保守主义，是在承认"我可能错、运气可能逆转、黑天鹅可能来"的前提下，仍然让财富增长。

- 守富 = 生存优先
- 只要不下牌桌，复利就能继续
- → 与第四章直接关联：中断 = 复利归零

## 形而上升维：成功 → 膨胀 → 坠落

你说的对——这不仅仅是金钱心理学，这是一个**跨领域的人类行为模式**：

### 希腊悲剧：Hubris → Nemesis

- **Hubris（傲慢）**：成功后的过度自信，认为自己超越了凡人的限制
- **Nemesis（报应）**：命运对傲慢的惩罚，通常以最残酷的形式降临
- 古希腊人早就看透了这个模式——不是神在惩罚你，是**傲慢本身改变了你的行为，行为导致了毁灭**

### 各领域的同构模式

| 领域 | 致富（成功） | 膨胀 | 坠落 |
|------|------------|------|------|
| **商业** | 公司做到行业第一 | "我们不可能被颠覆" | Kodak、Nokia、Blockbuster |
| **军事** | 打了几场胜仗 | 轻敌冒进 | 拿破仑征俄、希特勒征苏 |
| **体育** | 连胜纪录 | "我们不可战胜" | 被弱队爆冷 |
| **学术** | 发表重要论文 | 跨界发言、权威滥用 | 晚节不保 |
| **个人** | 升职加薪 | 消费升级、杠杆拉满 | 财务危机 |

**共同模式**：
1. 成功 = 能力 × 运气（当事人高估前者，忽略后者）
2. 成功后的环境变了，应对方式没变
3. 把"在某种条件下有效"当作"在所有条件下有效"

### 为什么人一定会膨胀？

这不是道德缺陷，是**认知机制**的问题：

- **自我归因偏误**：把成功归于自己，失败归于环境
- **控制幻觉**：成功后觉得一切都在掌控之中
- **幸存者确认**：身边都是成功者（失败者已经不见），强化"我做什么都对"
- **权力对大脑的物理影响**：研究发现权力会降低镜像神经元活跃度 → 难以共情 → 看不到风险信号

## 守富的具体方法论

### 1. 用"恐惧"保护财富

- 致富时恐惧是障碍，致富后恐惧是盔甲
- "如果这件事出错，我会不会死？"——这才是守富者该问的问题

### 2. 承认运气，保持谦逊

- 每次成功后问自己："这里有多少是运气？"
- 如果答案是"我至少被运气帮了 30%"——你会做出更保守的决策

### 3. 设置不可逆的底线

- 留一部分资产永远不碰（国债、现金、房产自住）
- 无论多膨胀，这部分资产不会动 → 坠落时还有反弹的资本

### 4. 身份与财富分离

- 膨胀的核心原因之一是**财富 = 身份**。财富缩水 = 身份崩塌 → 拼死守护 → 更冒险
- 如果你是谁不取决于你有多少钱，你就不会为了维护身份而做蠢事

## 关联知识

### 心理学

- **Dunning-Kruger 效应** — 刚获得一点成功的人最容易高估自己。"致富阶段"恰好处于"不知道自己不知道"的山顶
- **过度自信偏误（Overconfidence Bias）** — 80% 的司机认为自己技术高于平均水平。成功只会强化这种幻觉
- **热手谬误（Hot Hand Fallacy）** — 连胜之后觉得下一把也会赢，将随机序列误读为趋势

### 历史与政治

- **"修昔底德陷阱"的心理版本** — 不是国家之间的，是个人内部的：过去的成功成为未来失败的原因
- **权力悖论（Dacher Keltner）** — 获得权力的方式（共情、合作）和保持权力的方式（警惕、节制）截然不同，权力让人忘记后者

### 投资实践

- **Barbell Strategy（Taleb）** — 90% 极度保守 + 10% 极度冒险。守富用 90%，致富用 10%。互不侵犯
- **资产再平衡** — 机械地强迫自己"卖出赢家、买入输家"，用规则对抗膨胀

## 个人思考

> 阶段性成功 → 膨胀 → 摔跟头，这是一个轮回。

这个模式在个人层面几乎不可避免，因为：

1. 膨胀不是一种选择，是成功带来的**认知副作用**
2. 只有摔过跟头的人才会真正警惕——但那时候可能已经晚了

可能的解法：
- 不是"不膨胀"（做不到），而是**在膨胀的同时设置硬约束**——规则先行，不让膨胀的人做决策
- 类似 Ulysses 把自己绑在桅杆上：你知道自己会失控，所以在清醒时提前把自己锁住
- → 这就是"不可逆底线"和"资产再平衡"的意义：**不是因为你现在理性，而是因为你承认自己将来会不理性**
